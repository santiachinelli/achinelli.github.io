My Tech Stack

This is a non-exhaustive and constantly changing list of tools, ecosystems, and software I like to use.

I value stable, boring technology more than fadtech.

Infrastructure

  • Git (keeps my notes, configs and this website synced up)
  • Linux (server, desktop, embedded)
  • Unix (technically unix-likes, FreeBSD and MacOS)
  • gitea/gitlab-ce (two self-hosted git repositories I like for different reasons. gitlab-ce is better for onprem, gitea for small teams)
  • Docker (and podman)
  • Virtualization (QEMU and Virtualbox mainly, have had some fun with FreeBSD bhyve and jails in the past)
  • ZFS and BTRFS
  • Ansible (automation)
  • Bash (automation and general scripting fun)
  • Javascript (nodejs and currently using deno a lot)
  • Server hardening and best practices(learning container security practices since that's the way the industry has shifted)
  • Vue (and derivatives like Nuxt)

SaaS I like Using

  • Twilio (Telephony is a black box most of the time, much easier with Twilio)
  • Sendgrid (Also by Twilio, but much better/cheaper for newsletters than Constant Contact and other email marketing "tools"
  • AWS (When self-hosting is not an option)
  • Netlify(It's fast, although I also like Deno Deploy)

Fun Stuff

  • Hugo(and I guess, other SSGs) I started with Jekyll way back when, moved to Hugo for simplicity.
  • A-Frame (for building AR/XR experiences and generally having a good time)
  • Twine (Great for telling stories, surprisingly good as a presentation tool as well)
  • Judo (the sport, not a flavor-of-day frontend)
